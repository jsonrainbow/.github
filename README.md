# .github

This is the community health repository for the [JSON Rainbow](https://github.com/jsonrainbow) organization.

## About

This repository contains default community health files that apply to all repositories in the JSON Rainbow organization that don't have their own versions of these files.

## Contents

| File | Description |
|------|-------------|
| [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md) | Contributor Covenant Code of Conduct |
| [CONTRIBUTING.md](CONTRIBUTING.md) | Contribution guidelines |
| [SECURITY.md](SECURITY.md) | Security policy and vulnerability reporting |
| [FUNDING.yml](FUNDING.yml) | Sponsorship configuration |
| [ISSUE_TEMPLATE/](ISSUE_TEMPLATE/) | Issue templates (bug reports, feature requests) |
| [PULL_REQUEST_TEMPLATE.md](PULL_REQUEST_TEMPLATE.md) | Pull request template |
| [profile/README.md](profile/README.md) | Organization profile (displayed on org page) |

## How It Works

GitHub will use these files for any repository in the JSON Rainbow organization that doesn't have its own version of the file. This ensures a consistent experience across all our projects while allowing individual repositories to customize as needed.
